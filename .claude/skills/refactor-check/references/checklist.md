# 重构校验清单

## 必须通过（阻止项）

以下任一项不通过，**必须修复后才能上线**：

### 接口兼容性

- [ ] 函数签名未变（参数类型、数量、返回类型）
- [ ] 导出接口未变（export 列表）
- [ ] 类/接口的 public 成员未变

### 行为一致性

- [ ] 相同输入产生相同输出
- [ ] 边界情况处理一致（null、undefined、空值）
- [ ] 异常抛出行为一致（何时抛、抛什么）

### 金融逻辑（如适用）

- [ ] 资金计算结果一致
- [ ] 风控阈值判断一致
- [ ] 订单生成逻辑一致
- [ ] 手续费计算一致

## 警告项

以下项不通过需要**人工评估**：

- [ ] 新增了导出（可能是合理的扩展）
- [ ] 性能变化超过 10%
- [ ] 日志输出变化
- [ ] 内部实现细节变化（但外部行为不变）

## 快速校验流程

```bash
# 1. 检查函数签名变化
git diff HEAD~1 -- 文件路径 | grep -E "^[-+].*function|^[-+].*export"

# 2. 检查导出变化
git show HEAD~1:文件路径 | grep "^export" | sort > /tmp/before
grep "^export" 文件路径 | sort > /tmp/after
diff /tmp/before /tmp/after

# 3. 运行测试（如有）
pnpm test -- 文件路径

# 4. 类型检查
pnpm lint
```

## 金融代码特别检查

对于涉及资金的代码，额外检查：

- [ ] **精度问题**：浮点数计算是否使用了 Decimal 库
- [ ] **溢出问题**：大数计算是否有边界检查
- [ ] **四舍五入**：舍入逻辑是否与原来一致
- [ ] **币种处理**：多币种场景是否正确处理
- [ ] **时区问题**：时间相关逻辑时区处理是否一致
